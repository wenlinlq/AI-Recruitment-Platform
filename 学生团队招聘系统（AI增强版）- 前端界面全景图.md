## 一、界面概览

根据用户角色不同，系统包含**3大端、12个核心页面、20+功能组件**。以下是完整的界面地图：

```

│                   学生团队招聘系统界面地图                    │

│   学生端界面   │   团队管理端界面  │   管理员端界面            │
│  (6个核心页面) │   (4个核心页面)  │   (4个核心页面)           │
├───────────────┼─────────────────┼───────────────────────────┤
│  • 首页        │  • 团队仪表盘    │  • 系统总览仪表盘         │
│  • 招聘大厅    │  • 职位管理      │  • 用户管理              │
│  • 职位详情    │  • 候选人管理    │  • 团队管理              │
│  • 我的投递    │  • AI面试中心    │  • 系统配置              │

│  • AI面试模拟  │                 │                          │
└───────────────┴─────────────────┴───────────────────────────┘
```

## 六、响应式设计要点

### 桌面端（>1200px）
- 三栏布局，充分利用大屏
- 侧边菜单常驻
- 卡片网格展示

### 平板端（768px - 1200px）
- 两栏布局
- 侧边菜单可折叠
- 卡片宽度自适应

### 手机端（<768px）
- 单栏布局
- 底部导航栏
- 抽屉式菜单
- 卡片垂直排列
- 简化表格为卡片列表
